🛢️ Modelo Físico de Base de Datos

Visualiza y aprende el funcionamiento de la persistencia de datos relacionales en MySQL declarada en el archivo SQL del proyecto.

🔑 Clave Primaria (Primary Key - PK)

Es la columna que identifica de forma **única** a cada fila de una tabla. No se puede repetir y no puede ser vacía (nula). Ej: el campo id de la tabla de usuarios.

🔗 Clave Foránea (Foreign Key - FK)

Es una columna que sirve para enlazar o **referenciar** un registro de otra tabla. Ej: el campo created_by en documentos apunta al ID del usuario que lo creó.

🚀 Relación Intermedia (Muchos a Muchos)

Como un documento puede pertenecer a muchas colecciones y una colección tiene muchos documentos, creamos la tabla intermedia collection_documents que almacena pares de IDs.
