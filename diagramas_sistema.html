🛢️ Esquema Físico y Relacional de Datos (MySQL)

El sistema utiliza una base de datos centralizada en MySQL que mapea las relaciones de autenticación, almacenamiento de documentos, colecciones de investigación y sistema de comentarios estructurados. A continuación se detallan las tablas definidas físicamente en database/init_databases.sql.

👤 users
idINT (AUTO_INC)
emailVARCHAR(255) [UNIQUE]
password_hashVARCHAR(255)
nombreVARCHAR(255)
rolENUM('investigador', 'invitado')
institucionVARCHAR(255)
google_idVARCHAR(255)
avatar_urlVARCHAR(500)
is_activeBOOLEAN (DEFAULT TRUE)
created_atDATETIME
📄 documents
idINT (AUTO_INC)
tituloVARCHAR(255)
descripcionTEXT
autorVARCHAR(255)
file_pathVARCHAR(500)
is_publicoBOOLEAN
created_byINT (Ref: users.id)
visualizacionesINT (DEFAULT 0)
descargasINT (DEFAULT 0)
created_atDATETIME

idINT (AUTO_INC)

🔗 collection_documents
collection_idINT (Ref: collections.id)
